Instruction to upload Excel sheet

Please ensure your Excel sheet follows the structure and formatting rules outlined below:

Column Structure:
  1. Collage Code - Unique code for the college.
    Example: COL001
  2. Student Name - Full name of the student.
    Example: John Doe
  3. SR No - Serial number assigned to the student.
    Example: 12345
  4. Roll - Roll number of the student.
    Example: 6789
  5. Registration - Registration number for the student.
    Example: REG2024-001
  6. Season - Academic season or year.
    Example: 2024
  7. Semester - Name of the semester.
    Example: Spring
  8. Department - Field - Department or field of study.
    Example: Computer Science
  9. Probidhan - Name of the probidhan or regulation under which the student is registered.
    Example: Prob2024
  10. Shift - The shift the student is enrolled in (e.g., morning, evening).
    Example: Morning
Example Table Layout:
Collage Code Student Name SR No Roll Registration Season Semester Department - Field Probidhan Shift
COL001 John Doe 12345 6789 REG2024-001 2024 Spring Computer Science Prob2024 Morning
Or Download Demo Excel Sheet and Edit
File Format:

The Excel file must be saved with a .xlsx extension. Ensure there are no empty rows or columns between the data.

Uploading the File:
  1. Create the Excel sheet following the structure above.
  2. Save the file on your device.
  3. Go to the Add Students by uploading Excel Sheet page.
  4. Click on the "Choose File" button and select your Excel file.
  5. Click the "Add" button to upload and process the file.
